# DropMenu

A DropMenu appears based on either a click or hover. You can choose it's position relative
to the reference element.

    <DropMenu 
        control={() => <Button title="Menu" />}
        contents={() => (
            <Option>Apple</Option>
            <Option>Orange</Option>
            <Option>Banana</Option>
        )}
    />
