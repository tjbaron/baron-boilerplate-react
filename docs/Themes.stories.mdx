## Theme creation

The most important aspects of a Theme are:

1. color
2. font

### Color

To start experimenting with colors try using [Paletton](https://paletton.com/). Even
if you already have some colors you want to use or some brand guidelines to follow Paletton
will help you choose some complementary colors to fill out your palette.

To use your new theme export as XML and paste over the existing XML in `src/theme.ts`.
